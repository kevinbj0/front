css란

HTML이 웹 문서의 구조를 담당한다면
CSS 글꼴, 색상, 여백 같은 스타일을 추가함

css 적용방법 2가지

  • 내부 스타일 내부에 직접 집어넣는 style태그
  • 외부스타일, 링크

선택자

  • 전체선택자 *
  • 태그선택자 tag
  • class 선택자
  • 아이디선택자 # : 한 개
  • 자손 선택자 공백 : li .orange
  • li 태그이면서 orange 클래스를 가진 요소 어디든지
  • 자식 선택자 >
  • 인접형제선택자 +
  • 인접형제선택자 ~
  • 일치선택자 li.orange
  • li 태그이면서 orange 클래스를 가진 요소를 선택 -> 자손선택자랑 헷갈릴 수 있음
